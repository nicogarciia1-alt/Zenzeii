Quick summary for when you get to Block 4:

  POST /api/payments/create-checkout-session
  Auth: required (Bearer token)
  Body: { "tier": "premium" | "founding_member" }
  Response: { "checkout_url": "https://checkout.stripe.com/..." }

  To open payment page: window.location.href = checkout_url
  (Do not use router.push — it's an external Stripe URL)

  Success redirect lands at: /payment-success?session_id=...
  Cancel redirect lands at: /payment-canceled
  (Both pages need to be built in Block 4)
